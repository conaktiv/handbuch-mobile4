# Diagramm in Liste anzeigen

![Diagramm Übersicht](./diagramm-uebersicht.png#smartphone){ align=right }

Auch Diagramme lassen sich in Listen anzeigen; dies geschieht über die Auswahl von "Diagramm" als Feldtyp <span class="number">1</span>.

Über das Eingabefeld "Eigener Titel" <span class="number">2</span> lässt sich eine Beschriftung hinzufügen.

Aufgrund des Umfangs dieses Abschnitts werden die weiteren Optionen in der Reihenfolge ihres Auftauchens beschrieben.

<div class="clear"></div>

Die folgenden Abschnitte werden in Kürze ergänzt.

## Diagrammtyp festlegen

![Diagramm Typ](./diagramm-typ.png#smartphone){ align=right }

Unter "Voreinstellung" lässt sich per Klick auf die Schaltfläche <span class="number">1</span> der Typ des Diagramms auswählen.

Es öffnet sich ein Popupfenster <span class="number">2</span> mit den zur Verfügung stehenden Diagrammtypen.

<div class="clear"></div>

## Diagrammfelder verwalten

### Diagrammfelderliste

![Diagramm Felder Übersicht](./diagramm-felder-uebersicht.png#small){ align=right }

Unter Diagrammfelder werden die für dieses Diagramm verwendeten Felder in einer Liste aufgeführt und lassen sich bearbeiten.

In einem "frischen" Diagramm ist diese Sektion leer bis auf die Schaltfläche "+ Feld hinzufügen" <span class="number">1</span>.

<div class="clear"></div>

![Diagramm Felder Neu](./diagramm-felder-neu.png#small){ align=right }

Durch Klick auf diese Schaltläche wird ein leeres Feld eingefügt.

Um dessen Optionen anzuzeigen, muss es einmal angeklickt werden <span class="number">2</span>.

<div class="clear"></div>

![Diagramm Felder auswählt](./diagramm-felder-ausgewaehlt.png#small){ align=right }

Das ausgewählte Feld wird durch einen Haken rechts neben dem Feldnamen gekennzeichnet.

Es lässt sich nach oben oder unten schieben <span class="number">1</span>, womit die Darstellung im Diagramm beeinflusst wird.

Mit dem Stift <span class="number">2</span> lässt sich das Datenbankfeld auswählen.

Der Pinsel <span class="number">3</span> öffnet die Konfiguration für die Darstellung des Datenbankfelds.

Mit dem Mülleimer <span class="number">4</span> lässt sich das Feld entfernen.

<div class="clear"></div>

### Datenbankfeld für Diagramm auswählen

![Datenbankfeld Auswahl](./diagramm-dbfeld-uebersicht.png#medium){ align=right }

Ein Diagrammfeld kann entweder ein Listenfeld <span class="number">1</span> oder eine Systemvariable <span class="number">2</span> sein.

<div class="clear"></div>

#### Listenfeld auswählen

![Listenfeld ausgewählt](./diagrammfeld-dbfeld-ausgewaehlt.png#medium){ align=right }

Es können nur Felder im Diagramm angezeigt werden, die auch in der Listenstruktur vorhanden sind.

Durch Klick auf ein Feld <span class="number">1</span> wird dieses ausgewählt, was durch einen Haken rechts neben dem Feldnamen symbolisiert wird.

Dem Feld kann statt des vorgegebenen Namens auch ein eigener Name über das Eingabefeld bei "Eigener Titel" vergeben werden <span class="number">2</span>.

Dieser Titel ist mehrsprachfähig, d.h. durch Klick auf die Schaltfläche "de" neben dem Eingabefeld <span class="number">3</span> lassen sich Übersetzungstexte anlegen.

<div class="clear"></div>

#### Systemvariable auswählen

![Systemvariablen](./diagrammfeld-systemvariablen.png#small){ align=right }

Die ConAktiv® Mobile4 bietet neben den Listenfeldern auch Zugriff auf sogenannte Systemvariablen. Diese enthalten z.B. Urlaubsinformationen oder Zeitbuchungsinformationen.

Interessant werden diese Variablen dann, wenn man sie mit Listenwerten zusammenstellt. Enthält ein Listenwert z.B. die Information, wieviel Zeit der Mitarbeiter an einem Tag gebucht hat, lässt sich dies der Sollzeit pro Tag für diesen Mitarbeiter gegenüberstellen.

<div class="clear"></div>

#### Feld übernehmen

![Feld übernehmen](diagrammfeld-dbfeld-uebernehmen.png)

Durch Betätigen des Hakens rechts oben im Fenster <span class="number">1</span> wird das gewählte Feld als Diagrammfeld übernommen.

Um die Änderungen zu verwerfen, genügt ein Klick auf das "X" <span class="number">2</span> links oben im Fenster.

<div class="margin-bottom-large"></div>

![Feld übernommen](./diagrammfeld-dbfeld-eingefuegt.png#small){ align=right }

Wurde das Feld übernommen, wird es in der Diagrammfeldliste entsprechend dargestellt.

<div class="clear"></div>

### Darstellung des Diagrammfelds einstellen

![Formatierung Übersicht](./diagramm-dbfeld-formatierung-feld.png#medium){ align=right }

In der [Diagrammfelderliste](#diagrammfelderliste) lässt sich durch Betätigen des Pinsel-Symbols das Fenster für die Darstellungseigenschaften des Feldes öffnen.

Diese unterteilen sich in die Eigenschaften für Farben und Datensatz <span class="number">1</span> sowie die Darstellungsoptionen <span class="number">2</span>.

<div class="clear"></div>

#### Farben und Datensatz

![Formatierung Farben](./diagramm-dbfeld-formatierung-feld-farben.png#medium){ align=right }

Die Vorder- und Hintergrundfarbe für das Feld, so wie sie später im Diagramm erscheinen sollen, lässt sich frei einstellen.

Die Einstellung wird für die Vordergrundfarbe beschrieben, für die Hintergrundfarbe gelten die Schritte gleichermaßen.

Die aktuell gewählte Farbe wird unter "Gewählte Farbe" <span class="number">1</span> dargestellt.

Deren RGB (Rot/Grün/Blau)-Werte sind darunter zu sehen <span class="number">2</span> und lassen sich auch über die Texteingabefelder genau einstellen.

Um eine vordefinierte Farbe aus dem ConAktiv® Mobile4 Farbschema zu wählen, klicken Sie auf eine Farbe unter "Systemfarben" <span class="number">3</span>.

<div class="clear"></div>

![Formatierung Datensatz](./diagramm-dbfeld-formatierung-feld-datensatz.png#small){ align=right }

Über das Eingabefeld "Zuordnung zu Datensatz" lässt sich einstellen, ob das Feld für sich allein oder gruppiert mit anderen Feldern im Diagramm dargestellt werden soll.

<div class="clear"></div>

![Formatierung Datensatz Beispiel](./diagramm-dbfeld-formatierung-datensatz-bsp.png#small){ align=right }

In diesem Beispiel hat das Feld "Sollzeit" den Datensatz "default", den Feldern "Gebucht" und "Restzeit Anw." ist der Datensatz "default2" zugeordnet.

<div class="clear"></div>

#### Darstellungsoptionen

![Formatierung Darstellungsoptionen](./diagramm-dbfeld-formatierung-do.png#medium){ align=right }

Für das Feld lässt sich auch eine Darstellungsoption verwenden. Eine Liste der verfügbaren Optionen lässt sich durch Betätigen der Schaltfläche "---" <span class="number">1</span> anzeigen.

<div class="clear"></div>

![Formatierung Darstellungsoptionen Liste](./diagramm-dbfeld-formatierung-do-auswahl.png#medium){ align=right }

Die meistgenutzte Darstellungsoptionen an dieser Stelle dürfte "Zahl formatieren" sein.

<div class="clear"></div>

![Formatierung Darstellungsoptionen Zahl formatieren](./diagramm-dbfeld-do-zahlformatieren.png#medium){ align=right }

Hier wird eingestellt, dass der Zahlenwert mit Tausendertrennzeichen und zwei Nachkommastellen angezeigt werden soll.

<div class="clear"></div>

#### Darstellung übernehmen

![Darstellung übernehmen](diagrammfeld-dbfeld-formatierung-uebernehmen.png)

Durch Betätigen des Hakens rechts oben im Fenster <span class="number">1</span> werden die Darstellungseigenschaften für das Feld übernommen.

Um die Änderungen zu verwerfen, genügt ein Klick auf das "X" <span class="number">2</span> links oben im Fenster.

<div class="margin-bottom-large"></div>

### Wertebereich festlegen

![Wertebereich Übersicht](./diagramm-wertebereich-uebersicht.png#small){ align=right }

Über den Wertebereich lässt sich der Maßstab für die Diagrammfelder einstellen.

<div class="clear"></div>

#### Wertebereich "Höchster Wert"

Diese Einstellung ist sinnvoll, wenn mehrere Felder im Diagramm abgebildet werden, die in Relation zueinander gesetzt werden sollen (innerhalb jedes Listeneintrags).

#### Wertebereich "Absolut"

Soll dagegen ein Feld (oder auch mehrere) innerhalb der gesamten Liste vergleichbar gemacht werden, empfiehlt sich die Angabe eines absoluten Höchstwerts, zu dem das Feld/die Felder skaliert werden.

![Wertebereich absolut](./diagramm-wertebereich-absolut-uebersicht.png#smartphone){ align=right }

Der absolute Wertebereich reicht von einem Von-Wert <span class="number">1</span> bis zu einem Bis-Wert.

Der Bis-Wert kann entweder ein absoluter Wert sein <span class="number">2</span> oder eine Systemvariable <span class="number">3</span>.

Füllt ein Feld nicht den kompletten Wertebereich aus, lässt sich eine Füllfarbe für den verbleibenden Bereich einstellen, die sich in die Füllfarbe <span class="number">4</span> und die Rahmenfarbe <span class="number">5</span> aufgliedert.

<div class="clear"></div>

##### Systemvariable für Wertebereich wählen

![Wertebereich Systemvariable wählen](./diagramm-wertebereich-absolut-systemvariable-button.png#small){ align=right }

Unter dem Tab "Systemvariable" lässt sich durch Betätigen der Schaltfläche "..." eine Systemvariable wählen, die als Wert für das Bis-Feld genutzt wird.

<div class="clear"></div>

![Wertebereich Systemvariable Auswahl](./diagramm-wertebereich-systemvariable-waehlen.png#medium){ align=right }

In dem erscheinenden Popup lässt sich in der Liste der Systemvariablen <span class="number">1</span> die gewünschte auswählen.

Bei Bedarf kann ein eigener Titel über das entsprechende Eingabefeld <span class="number">2</span> eingegeben werden.

Durch Betätigen des Hakens rechts oben im Fenster <span class="number">3</span> wird die Systemvariable als Wertebereich Bis übernommen.

Um die Änderungen zu verwerfen, genügt ein Klick auf das "X" <span class="number">4</span> links oben im Fenster.

<div class="clear"></div>

### Diagrammkomponenten ein-/ausblenden

![Diagramm Komponenten](./diagramm-komponenten.png#small){ align=right }

Verschiedene Bestandteile des Diagramms lassen sich ein- und ausblenden. Dazu gehören die Legende und das Koordinatensystem.

Außerdem lässt sich steuern, ob die Werte der Felder im Diagramm selbst (in einem Balkendiagramm z.B. innerhalb der Balken) angezeigt werden sollen, oder nur bei Überfahren mit der Maus.

<div class="clear"></div>

### Farben festlegen

![Diagramm Farben](./diagramm-farben.png#small){ align=right }

Es lassen sich verschieden Farbwerte des Diagramms einstellen.

Die Hintergrundfarbe ist die flächige Füllfarbe hinter dem Diagramm.

Die Beschriftungsfarbe regelt die Beschriftung der Legende und des Koordinatensystems.

Die Rasterfarbe regelt die Darstellung der Achsenunterteilungen.

<div class="clear"></div>

![Diagramm Farbe auswählen](./Diagramm-farbe-auswaehlen.png#small){ align=right }

Durch Betätigen einer der "Farbe"-Schaltflächen öffnet sich ein Popup zur Farbauswahl.

Die aktuell gewählte Farbe wird unter "Gewählte Farbe" <span class="number">1</span> dargestellt.

Deren RGB (Rot/Grün/Blau)-Werte sind darunter zu sehen <span class="number">2</span> und lassen sich auch über die Texteingabefelder genau einstellen.

Um eine vordefinierte Farbe aus dem ConAktiv® Mobile4 Farbschema zu wählen, klicken Sie auf eine Farbe unter "Systemfarben" <span class="number">3</span>.

Um die Farbänderungen zu übernehmen, betätigen Sie den Haken rechts oben im Fenster <span class="number">4</span>.

Um die Änderungen zu verwerfen, klicken Sie auf das "X" links oben im Fenster <span class="number">5</span>.

<div class="clear"></div>

### Basiswert festlegen

![Basiswert festlegen](./diagramm-basiswert.png#smartphone){ align=right }

Der Basiswert wird auf den Wert aller Diagrammfelder addiert. Er dient in erster Linie dazu, Nullwerte zu visualisieren.

<div class="clear"></div>

## Abmessungen des Diagramms

![Diagramm Abmessungen](./diagramm-abmessungen.png#small){ align=right }

Für Diagramme in Listen muss zwingend eine Höhe angegeben werden. Diese beträgt in der Voreinstellung 100 Pixel, kann aber frei geändert werden.
